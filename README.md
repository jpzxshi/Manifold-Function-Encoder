# Manifold-Function-Encoder
Manifold Function Encoder: Identifying Different Functions Defined on Different Manifolds (arXiv:2512.20227)  

Environment: python=3.12.9 numpy=2.3.1 pytorch=2.6.0+cu126  

1. Run 'generate_star_domains.py' for generating raw star domain data.  
2. Run 'generate_annular_domains.py' for generating raw annular domain data.  
3. Run 'data.py' for integrating into training data.  
4. Run 'main.py' for training model.
